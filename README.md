# 🏡 Cascade Valley Painting — Business Website

A fully custom, professionally designed business website built from scratch for **Cascade Valley Painting**, a residential and commercial painting company serving the Pacific Northwest.

**🌐 Live Site:** [cascadevalleypainting.com](https://cascadevalleypainting.com)

---

## 📋 Overview

This project showcases end-to-end business website development — from design concept to live deployment.

Built entirely with vanilla HTML, CSS, and JavaScript, the site delivers a polished, modern brand presence optimized for both desktop and mobile users.

---

## ✨ Features

- **Responsive Design** — Looks sharp on desktop, tablet, and mobile
- **Smooth Scroll Navigation** — Fixed nav with active section highlighting
- **Service Showcase** — Clearly structured sections highlighting painting services
- **Professional Typography** — Playfair Display + DM Sans pairing for an upscale feel
- **Contact & Call-to-Action Sections** — Designed to convert visitors into leads
- **Performance Optimized** — No frameworks, no bloat — fast load times
- **Local SEO Ready** — Structured for Pacific Northwest search visibility
- **Custom Color Branding** — Forest green, navy, and warm accent palette matching the company identity
